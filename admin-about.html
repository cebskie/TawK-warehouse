TawK Warehouse Management System

TawK is a streamlined warehouse management system designed to simplify stock tracking, product transactions, and category management. Built with an intuitive interface, TawK allows users to efficiently monitor inventory levels, record transactions, and manage product details in real time. Whether for small businesses or larger operations, TawK ensures accurate inventory control, reducing errors and improving operational efficiency!


Here are the admin features:

1. Transaction Log

To check a transaction performed by a user, click the transaction button in the navigation bar. You’ll see a table of 10 rows of the most recent transactions done by the users. You can then prepare the product to be taken for out-type transactions, or prepare the storage for in-type transactions.


2. Register User

Before a user has access to check products stock and perform transactions, they need to be registered to the system. You can do that by clicking the user button in the navigation bar and type the user’s name, address, and phone number. Once you make sure everything is correct, click the button to add the user to the system. You can also search for any user by the user id or user name. Click the data button, select the search type, then type in the user id or user name. The user’s details will be shown in a table once you click the search button.


3. Add Product

If there’s a new product, that has never been registered in product database, you must input the product details first. To add new product, click the product button in the navigation bar, then you can select the category of new product, type the new product’s name, and the price. Once you make sure the details are right, click on the button to add the product to the database.


4. Check Stock

To check a product’s stock quantity, just scroll down the stock button in the product page and you’ll be directed to the check stock page. There, you can select the product category and product name. You can then click the button to check the product stock. There will be a table that shows the product name, stock quantity, and product price.


5. View Insights

By clicking the insight button in the navigation bar, you can see some reports of the products, such as the stock movement, most popular products, low stock products, and dead stock.
